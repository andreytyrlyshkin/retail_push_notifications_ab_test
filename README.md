## Описание проекта

**Контекст:** Крупная розничная сеть (ретейлер) рассматривает возможность замены баннеров в приложении на пуш-уведомления в качестве основного канала маркетинговых коммуникаций. 
Отдел маркетинга предполагает, что новый формат может положительно повлиять на ключевые метрики: **конверсию в покупку** и **средний чек**.

**Цель исследования:** Провести анализ результатов A/B-теста, чтобы оценить влияние новой механики на поведенческие метрики клиентов и 
предоставить руководству обоснованные рекомендации по внедрению изменений, подкрепленные финансовой моделью.

---

## Ход работы

Проект включает в себя полный цикл анализа данных, от ETL до построения прогнозного калькулятора в Excel.

1.  **Загрузка и первичный анализ данных (EDA):**
    *   Импорт и объединение трех таблиц: данные о клиентах, платежах и регионах.
    *   Проверка на пропуски и дубликаты. Принято решение об удалении строк с пропусками (< 6% от общего объема данных).
    *   Изучение распределения торговых точек по городам.

2.  **Предобработка и расчет метрик:**
    *   Агрегация платежей на уровне клиента (суммарный чек за период эксперимента).
    *   Объединение платежей с данными о клиентах и регионах.
    *   Создание бинарного флага `is_paid` для расчета конверсии.

3.  **Статистический анализ результатов A/B-теста:**
    *   **Общий анализ:** Сравнение средних платежей (t-критерий Стьюдента) и распределений (критерий Манна-Уитни) в тестовой и контрольной группах. Анализ конверсии.
    *   **Сегментация:** Детальный анализ результатов в разрезе городов и отдельных торговых точек для выявления сегментов, показавших наилучший результат.

4.  **Разработка финансового калькулятора (Excel):**
    *   Создан инструмент для моделирования экономического эффекта от внедрения новой механики.
    *   Калькулятор позволяет варьировать количество клиентов (`N`) и оценивать совокупный прирост или падение выручки на основе статистически значимых результатов по каждой торговой точке.

## Используемые инструменты

*   **Python (Jupyter Notebook):**
    *   `pandas`, `numpy`: обработка и трансформация данных.
    *   `scipy`: проведение статистических тестов (Т-критерий Стьюдента, критерий Манна-Уитни).
    *   `matplotlib`, `seaborn`: визуализация.
    *   `pandas`, `xlsxwriter`: экспорт итоговых таблиц в Excel.
*   **Excel:** построение калькулятора для оценки совокупного эффекта на выручку.
*   **Git:** контроль версий.

## Результаты и ключевые выводы

*   **На уровне всей выборки:** Новая механика (пуш-уведомления) показывает статистически значимый прирост как в среднем чеке, так и в конверсии по сравнению с контрольной группой (баннеры).
*   **Региональная сегментация:** Выявлена неоднородность эффекта:
    *   Наибольший положительный эффект зафиксирован в городах-миллионниках (Москва, Санкт-Петербург, Самара).
    *   В ряде регионов статистически значимого эффекта обнаружено не было, что может говорить о недостаточной мощности выборки или особенностях поведения пользователей.
*   **Экономическая оценка:** Разработанный калькулятор позволяет рассчитать потенциальный прирост выручки при масштабировании механики на определенное количество пользователей.
    *   *Пример:* При масштабировании на 10 000 пользователей прогнозируемый положительный эффект покрывает затраты на разработку (настройку механики), что делает реализацию экономически целесообразной.

## Как воспроизвести проект

1.  Клонируйте репозиторий.
2.  Установите зависимости при необходимости.
3.  Откройте и выполните Jupyter Notebook `notebooks/ab_test_analysis.ipynb`.
4.  Для расчёта выручки по параметрам откройте файл `results/ab_test_calculator.xlsx`.

## Структура репозитория

*   `data/`: Исходные данные для анализа.
*   `notebooks/`: Jupyter Notebook с полным циклом анализа.
*   `results/`:
    *   `ab_test_results.xlsx` — технический файл с "сырыми" результатами теста, экспортированный из Jupyter Notebook (перезаписывается при повторном запуске кода).
    *   `ab_test_calculator.xlsx` — финальный Excel-калькулятор с финансовой моделью, построенный на основе данных из `ab_test_results.xlsx`.
*   `requirements.txt`: список необходимых Python-библиотек.
*   `LICENSE`: лицензия MIT.
*   `.gitignore`: список исключаемых из Git файлов.
